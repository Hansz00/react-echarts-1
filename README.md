[![Dependency Status][deps-badge]][deps]
[![devDependency Status][dev-deps-badge]][dev-deps]

## Installation

```sh
npm install react-echarts --save
```

## Usage

```js
import ECharts from 'react-echarts';

<ECharts option={this.state.option} />
```
